About This Project

TravelMate is a comprehensive travel planning web application designed to provide users with personalized itineraries, nearby services, and intelligent city suggestions. Our platform leverages modern web technologies and AI integration to create seamless travel experiences.

Key Features:

  • AI-powered travel plan generation with detailed day-by-day itineraries
  • Smart city autocomplete with 300+ Indian destinations
  • Nearby restaurants, hotels, and shopping recommendations
  • Interactive maps integration for directions
  • Budget estimation and cost breakdown
  • Responsive design for all devices

Technologies Used:

  • Backend: Flask, Python, Requests
  • Frontend: HTML5, CSS3, JavaScript
  • APIs: Google Gemini AI, Google Maps
  • Styling: Custom CSS with modern design principles
  • Environment: Python-dotenv for configuration

This project demonstrates the integration of AI capabilities with traditional web development to create intelligent, user-friendly travel planning solutions.
